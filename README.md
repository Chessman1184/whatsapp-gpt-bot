# WhatsApp ChatGPT Bot

A simple WhatsApp bot that uses **ChatGPT API** to reply to your messages automatically.  
Built with [`whatsapp-web.js`](https://github.com/pedroslopez/whatsapp-web.js) and OpenAI's ChatGPT API.

---

## Features

- Connect your WhatsApp account by scanning a QR code.
- Chat naturally with ChatGPT AI on WhatsApp.
- Easy to deploy on platforms like Railway, Render, or locally.
- Supports GPT-4o or GPT-3.5-turbo models.
- Session is saved locally to avoid repeated QR scans.
